## 截斷高度收斂?

改截斷高度測試收斂性(convergence_H.m)

```mermaid
graph LR
A(設定截斷高度區間) -->B[改h]
    B --> C[改N]
    C --> F[解流場]
    F --> D{特徵值收斂}
    D -->|是| E[紀錄特徵值&N]
    D -->|否| C
    E --> G{測試完成}
    G -->|是| H(畫圖)
    G -->|否| B
```

判斷特徵值收斂 : 前後項相對誤差$$<10^{-10}$$

#### 結果:

以$$k=3.2$$為例，在Zhang(1996)應該是stable。

截斷高度大於兩倍波長之後收斂到$$10^{-12}$$。

選$$h = 2\lambda,\ N = 650$$。
